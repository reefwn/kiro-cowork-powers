# Literature Review

Search biomedical literature and preprints, access full-text publications, and synthesize findings.

## Workflow

### 1. Define Search Strategy
- Clarify the research question with the user
- Identify key terms, MeSH headings, and Boolean operators
- Determine date range and scope (reviews, primary research, preprints)

### 2. Search Sources
- Search ~~literature (PubMed) for peer-reviewed articles
- Search ~~literature (bioRxiv/medRxiv) for recent preprints
- Access full-text through ~~journal access when available
- Check ~~data repository for associated datasets

### 3. Screen and Filter
- Review titles and abstracts for relevance
- Prioritize by impact factor, citation count, and recency
- Flag key methodological papers vs. review articles
- Note any conflicting findings across studies

### 4. Synthesize Findings
- Summarize key themes and consensus findings
- Highlight gaps in the literature
- Identify methodological trends
- Note potential collaborators or competing groups

### 5. Create Figures
- Use ~~scientific illustration to create summary diagrams
- Generate graphical abstracts or pathway diagrams as needed

## Output Format

Provide a structured summary with:
- Search strategy used (terms, filters, databases)
- Number of results screened
- Key findings organized by theme
- Critical papers with brief annotations
- Identified gaps and suggested next steps
